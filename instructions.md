You are a senior frontend developer and UX designer.

Task: Create a complete, production-ready, mobile-first restaurant website.

Website type: Local restaurant business website

Design philosophy:
- Clean, minimal, food-focused design
- High contrast CTAs
- Mobile-first, responsive layout
- Fast loading, no unnecessary animations
- Apple-like frosted glass where appropriate
- Warm, premium restaurant vibe

Structure the website in the following order:

1. HERO SECTION
- Restaurant name, short tagline
- Primary CTA: “Order Now” or “Reserve Table”
- Secondary CTA: “View Menu”
- Background image with dark overlay for readability

2. ABOUT SECTION
- 3–4 lines about the restaurant
- Cuisine type, years in service, specialty

3. MENU SECTION
- Categorized menu (Starters, Mains, Desserts, Drinks)
- Each item: name, short description, price
- Veg / Non-veg indicator
- Grid layout optimized for mobile

4. CHEF SPECIAL / HIGHLIGHTS
- 2–3 featured dishes
- Larger images and short description

5. ONLINE ORDERING / RESERVATION
- Prominent CTA buttons
- Option to link WhatsApp / Zomato / Swiggy
- Simple reservation form (name, phone, date, time)

6. TESTIMONIALS / REVIEWS
- 3 customer reviews
- Star ratings
- Trust-building layout

7. GALLERY
- Responsive image grid
- Lazy-loaded images

8. LOCATION & CONTACT
- Address
- Google Maps embed
- Click-to-call phone number
- Business hours

9. FOOTER
- Social media links
- Copyright
- Small branding note

Technical requirements:
- Semantic HTML
- Clean CSS (Flexbox / Grid)
- No external frameworks
- Optimized images
- Accessibility friendly
- Commented code
- Separate files: index.html, style.css, script.js

After generating code:
- Explain how to customize content
- Explain how to deploy on Netlify
- Explain how to adapt this template for another restaurant
