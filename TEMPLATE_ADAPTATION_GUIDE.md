# Template Adaptation Guide

## Overview

This guide explains how to adapt this restaurant website template for another restaurant or food business.

## Quick Adaptation Checklist

- [ ] Update restaurant name and branding
- [ ] Replace logo and images
- [ ] Customize color scheme
- [ ] Update menu items and prices
- [ ] Modify contact information
- [ ] Adjust business hours
- [ ] Update social media links
- [ ] Customize hero tagline
- [ ] Change testimonials
- [ ] Update Google Maps location

For detailed instructions on each customization, see CUSTOMIZATION_GUIDE.md

## Restaurant Type Adaptations

### Color Schemes by Cuisine

**Italian Restaurant:**
- Primary: #C8102E (Italian Red)
- Accent: #009246 (Italian Green)

**Japanese Restaurant:**
- Primary: #1A1A1A (Black)
- Accent: #C41E3A (Red)

**Mexican Restaurant:**
- Primary: #D4AF37 (Gold)
- Accent: #E63946 (Red)

**Cafe/Bakery:**
- Primary: #6F4E37 (Coffee Brown)
- Accent: #F4A460 (Sandy Brown)

### Menu Category Adaptations

**Pizza Restaurant:** Pizzas, Pastas, Salads, Drinks
**Cafe:** Coffee, Breakfast, Lunch, Pastries
**Sushi Bar:** Sushi, Sashimi, Rolls, Appetizers

## Quick Start

1. Update `style.css` color variables (lines 20-30)
2. Replace all images in `images/` folder
3. Update menu items in `index.html`
4. Change contact information
5. Test on mobile and desktop

For complete adaptation instructions, see the full guide in CUSTOMIZATION_GUIDE.md
